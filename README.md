## 🖥️ What is markdown-preview?

markdown-preview is a simple app that lets you write and see Markdown text right away. Markdown is a way to format text using plain typing rules. Instead of complex menus, you type easy commands like `#` for headings or `*` for lists. This tool shows your formatted text as you write. It helps to create notes, documents, or posts faster without switching screens.

This app works on Windows computers. You do not need to know programming or how Markdown code works. Just type and watch your writing come to life.

### Basic Markdown tips

Here are some common commands for Markdown you can try:

- `# This is a Heading` for big titles.
- `## This is a smaller heading` for subtitles.
- `* Item 1` to start a bullet list.
- `1. Item 1` to start a numbered list.
- `**bold**` for bold text.
- `*italic*` for italic text.
- `[Link text](https://example.com)` to add a clickable link.
- `![Alt text](image-url.jpg)` to add images.

You do not need to remember them all. The app comes with examples and help to get started.
